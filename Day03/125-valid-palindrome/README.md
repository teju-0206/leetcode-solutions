# Valid Palindrome (LeetCode #125)

## Approach
- Brute Force: Build cleaned string → reverse → compare
- Optimized: Two pointers, skip non-alphanumeric and compare in-place

## Notes
- Ignoring spaces and symbols is key
- Case-insensitive comparison required
- Two-pointer approach avoids extra space

## Complexity
- Brute Force: O(n²), O(n)
- Optimized: O(n), O(1)
