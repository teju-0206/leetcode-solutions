# Day 01

## Problems Solved
- Two Sum (LeetCode #1)
- Contains Duplicate (LeetCode #217)

---

## Approaches

### Two Sum
- Used HashMap to store value → index
- Checked if complement exists while iterating

### Contains Duplicate
- Used HashSet to track seen elements
- Returned true if element already exists

---

## Key Learnings
- HashMap for fast lookup (value → index)
- HashSet for detecting duplicates
- Pattern: check first, then store

---
